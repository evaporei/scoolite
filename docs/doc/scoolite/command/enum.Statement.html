enum Statement {
    Insert(String),
    Select,
}

Statement is the enum that contains all of the statements for scoolite. An example of a statement is insert, it does belong to the SQL specification and it is used to add a row to a table.

Tries to parse an &str input into a Box<Command>, if it fails it returns a "not implemented error" Error.

fn insert(&self, input: &str, table: &mut Table) -> Result<(), Error>
[src]

Creates a new Row based of an input &str and inserts it inside of a table. This is what get's called when something like Statement::Insert("insert 1 john john@mailbox.com").execute() happens.

fn select(&self, table: &Table) -> Result<(), Error>
